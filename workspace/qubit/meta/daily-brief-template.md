# Daily Brief Template (Decision+Action Seed)

Use this structure for each pillar daily brief.

1. Decisions Needed
List decisions that unblock progress.

2. Due Soon
List reminders due in the next 48 hours.

3. Blocked Projects
List blocked projects only.

4. Top 3 Next Actions
List highest-leverage next actions.

5. Focus Question
Ask one question that drives immediate action.
