# Policy: Risk and Autonomy

## Default

1. Auto-apply low-risk inferred actions.
2. Require clarification for high-risk inferred actions.
3. Treat explicit `qubit ...` commands as authorized execution.

## Low-Risk Inferred Actions

1. Journal append
2. Reminder append
3. Contact note append

## High-Risk Inferred Actions

1. Project creation inferred from ambiguous phrasing
2. Structural file changes not explicitly requested
3. Any action with low confidence and multi-interpretation ambiguity

## Clarification Rule

Ask one concise question with 2-3 options and stop.
